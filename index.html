Spaceships Online

A Project by Stefan Kussmaul

Written from scratch in pure Javascript, with Node.js backend

Use 'A' and 'D' to rotate, 'W' to accelerate forward, and 'S' to brake. Use 'Space' to shoot bullets.

This game is in really early stages! Multiplayer is kind of working, but gameplay is currently choppy due to lag (Something I'm working on addressing)

Also, I'm still working on the lobbying code, so there may be some strange behavior when one or more players leave...

This project is currently still in early stages. A lot of the backbone is there, but there's still a lot to fix as the project matures.
